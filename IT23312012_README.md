## How to Run the Project

1. Open terminal inside project folder  

2. (Optional) Activate virtual environment  
   venv\Scripts\activate  

3. Install dependencies  
   pip install -r IT23312012_requirements.txt  
   playwright install  

4. Run the automation script  
   python IT23312012_test_automation.py --excel "IT23312012_Test_cases.xlsx" --url "https://www.pixelssuite.com/chat-translator"
